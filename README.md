# Real Estate Property Management System

## Overview
A full-stack web application that allows users to browse properties for rent or purchase, and enables owners or agents to list properties for sale or rent. Buyers and renters can send inquiries, and owners or agents can respond, streamlining property management and communication. The platform also includes subscription-based features for enhanced access and monetization.

## Features
- **User Functionality**
  - Browse properties by category (rent or buy)
  - Search and filter listings by location, price, and property type
  - Send inquiries to owners or agents directly from property pages
  - Unlock owner/agent contact details using a subscription plan

- **Owner/Agent Functionality**
  - Post up to **2 properties for free**
  - Choose from subscription plans to post additional properties beyond the free limit
  - Manage property listings (edit or delete)
  - Respond to buyer/renter inquiries via integrated messaging system

- **Admin/General Features**
  - User authentication and role-based access control
  - Manage Reviews 
  - Responsive and intuitive UI for easy navigation
  - Real-time notifications for inquiries and responses
  - Subscription management system for monetization

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript 
- **Backend:** / Django   
- **Database:** SQLite  
- **Hosting/Deployment:** (e.g., Heroku, Vercel, AWS)
